# AWS Cloud Governance & Automation Platform

A serverless AWS project that automatically scans cloud resources for security and governance issues.

## Services Used

- AWS Lambda
- EventBridge
- SNS
- DynamoDB
- CloudWatch
- IAM
- EC2
- Python (Boto3)

## Features

- EC2 Instance Auditing
- Security Group Auditing
- Automated Alerts
- Findings Storage
- Scheduled Scans
